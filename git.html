Git Hub

GitHub é uma plataforma de desenvolvimento colaborativo de software baseada em nuvem. Funciona com o sistema de controle de versão Git e oferece uma variedade de recursos para gerenciar projetos de software e facilitar a colaboração entre desenvolvedores. Os principais recursos incluem hospedagem de repositórios Git, controle de versão para rastrear alterações no código-fonte, colaboração através de solicitações de recebimento e discussões de problemas, gestão de projetos com ferramentas como quadros kanban e marcos, e integração com ferramentas de integração contínua e implantação contínua.

Qual sua função?

O Git Hub é Utilizadao para o armazenamento de códigos e scripts, utilizando a função de repositório você poderá guardar todos arquivos e códigos do seu site

Repositórios

Os repositórios no GitHub são como pastas onde os projetos de software são armazenados e gerenciados. Dentro de um repositório, você encontrará todos os arquivos relacionados ao projeto, como o código-fonte, a documentação, os arquivos de configuração e muito mais. Cada repositório possui uma estrutura organizada que geralmente inclui arquivos de código-fonte, um arquivo README.md para fornecer informações sobre o projeto, uma licença que define os termos de uso do software e outros recursos relevantes. Os repositórios são essenciais para colaboração entre desenvolvedores, permitindo que eles trabalhem em conjunto em um projeto, proponham alterações, revisem o trabalho um do outro e controlem as versões do código-fonte ao longo do tempo. Além disso, os repositórios são usados para distribuir software, documentar o projeto, rastrear problemas e gerenciar solicitações de recursos. Em resumo, os repositórios do GitHub são o coração de qualquer projeto de software hospedado na plataforma, proporcionando um espaço centralizado para desenvolvimento, colaboração e distribuição de software.
